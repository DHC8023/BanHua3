DEMO
===========================

###########环境依赖
autoJs 4.1.1+
autoJs Pro v7.0+
坚果云网盘

###########部署步骤
1. 注册坚果云网盘
    地址：https://www.jianguoyun.com
2. 注册泡椒云
    地址：http://paojiaoyun.com

3. 安装相关vscode插件

4. 获取坚果云相关数据
    账号
    密钥
    该项目路径


###########目录结构描述
{
    "mod"               "脚本模块存放文件夹",
    "main.js"           "脚本客户端入口地址",
    "mainActivity.js"   "脚本云端入口地址",
    "script.js"         "主要脚本文件"
}
