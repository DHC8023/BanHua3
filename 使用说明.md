DEMO
===========================

###########环境依赖
autoJs 4.1.1+
autoJs Pro v7.0+
坚果云网盘

###########部署步骤
1. 注册坚果云网盘
    地址：https://www.jianguoyun.com

2. 安装相关vscode插件

3. 获取坚果云相关数据
    账号
    密钥
    js文件存放路径


###########目录结构描述

├── mod                         // js模块
├── mainActivity.js             // 服务端入口
│── script.js                   // 主要脚本
└── main.js                     // 客户端入口
